# 🌤️ Weather Application

A modern weather application that displays real-time weather data using API integration. The app supports city-based search and automatic location detection.

---

## 🚀 Features

* 🔍 Search weather by city
* 📍 Auto-detect current location
* 🌡️ Real-time temperature display
* 💧 Humidity and 🌬️ wind speed
* 🌈 Dynamic background based on weather
* ⌨️ Enter key support
* ⏳ Loading state handling
* 🎨 Modern UI (glassmorphism design)

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Weather API

## 💡 How It Works

* User enters city or uses location
* JavaScript sends request to weather API
* API returns weather data in JSON format
* UI updates dynamically using DOM manipulation
